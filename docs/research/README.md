# Research Writeups

Each Phase 1 spike (see PLAN.md) lands its findings here as a short markdown file:

- `rpcs3-control.md` — how we drive the emulated portal dialog.
- `rpcs3-launch.md` — process launch, version detection, serial-based boot.
- `firmware-inventory.md` — indexer output + classification for the `.sky` pack.
- `wiki-scrape.md` — Fandom scrape plan, hit rate, attribution.
- `stack-smoke.md` — Axum + Leptos + egui coexistence check.

Keep them short — capture the decision, the evidence, and the risks. Detailed design decisions live in CLAUDE.md; ephemeral exploration lives here.
